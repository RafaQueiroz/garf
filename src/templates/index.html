Descrição

GARF é um gerador automatizado de regras de firewall. Este baseia-se em dados coletados por servidores Honeypot e configurações definidas pelo usuário para gerar as regras que serão aplicadass no firewall. A ferramenta usa as seguintes tecnologias:

  • FileBeat
  • Logstash
  • Elasticsearch
  • Flask
  • Spectre css
